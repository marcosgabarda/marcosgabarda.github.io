fastfetch.
Posted on 2025-06-08 :: 39 Words :: Tags: debian

I use fastfetch as alternative to neofetch to gather system information.

  1. Download the latest .deb package released from here.

  2. Install the package:

sudo apt install ./fastfetch-linux-amd64.deb
